# 07 — I2C Test

## Objective

Initialize I2C1 bus and perform a device scan. Verify bus initialization and basic read/write capability.

## Board-Specific Pin Map

| Function | Pin | Notes |
|----------|-----|-------|
| I2C1 SCL | PB8 | Arduino D15, CN5 pin 10 |
| I2C1 SDA | PB9 | Arduino D14, CN5 pin 9 |

## Test Script

Generate `tests/<PREFIX>_test_i2c.py`:

```python
from machine import I2C, Pin
import time

# Test 1: I2C1 hardware init
try:
    i2c = I2C(1, freq=400000)
    print("PASS: I2C1 init at 400 kHz")
except Exception as e:
    # Fallback: try with explicit pins
    try:
        i2c = I2C(1, scl=Pin('PB8'), sda=Pin('PB9'), freq=400000)
        print("PASS: I2C1 init with explicit pins at 400 kHz")
    except Exception as e2:
        print("FAIL: I2C1 init: {}".format(e2))
        print("I2C_TEST_COMPLETE")
        raise SystemExit

# Test 2: Bus scan
try:
    devices = i2c.scan()
    print("I2C1 scan: {} device(s) found".format(len(devices)))
    if devices:
        for addr in devices:
            print("  Device at 0x{:02X} ({})".format(addr, addr))
        print("PASS: I2C1 scan found devices")
    else:
        print("WARN: I2C1 scan found no devices (none connected — this is OK)")
        print("PASS: I2C1 scan completed (bus functional)")
except Exception as e:
    print("FAIL: I2C1 scan: {}".format(e))

# Test 3: I2C write to non-existent address (should raise OSError)
try:
    i2c.writeto(0x00, b'\x00')
    print("WARN: I2C1 write to 0x00 did not raise error")
except OSError as e:
    print("PASS: I2C1 write to invalid addr raises OSError (expected)")
except Exception as e:
    print("FAIL: I2C1 unexpected error on write: {}".format(e))

# Test 4: SoftI2C fallback test (uses bitbanging — works on any GPIO pair)
try:
    from machine import SoftI2C
    si2c = SoftI2C(scl=Pin('PB8'), sda=Pin('PB9'), freq=100000)
    devices_soft = si2c.scan()
    print("SoftI2C scan: {} device(s)".format(len(devices_soft)))
    print("PASS: SoftI2C init and scan")
except ImportError:
    print("SKIP: SoftI2C not available in this build")
except Exception as e:
    print("FAIL: SoftI2C: {}".format(e))

print("I2C_TEST_COMPLETE")
```

## Execution

```bash
python scripts/pyboard.py --device <COM_PORT> -f cp tests/<PREFIX>_test_i2c.py :
python scripts/pyboard.py --device <COM_PORT> tests/<PREFIX>_test_i2c.py
```

## Verification

- **Pass**: Output contains `I2C_TEST_COMPLETE` and `PASS: I2C1 init`
- The scan finding 0 devices is acceptable if no I2C peripherals are connected
- **Fail**: If I2C(1) is not recognized, try I2C(0) or explicit SoftI2C
- **Retry strategy**: Switch to `SoftI2C` with explicit pins as fallback

## Optional: I2C Device Test

```python
addr = 0x76  # Change to detected address

```
